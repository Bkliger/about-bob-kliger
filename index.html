ÖVA

ÖVA is a run coaching application. An athlete enters their planned weekly training mileage and the days of the week they plan to run, and ÖVA generates a plan based on previous efforts which are gathered through tight integration with Strava. This project was built using an Angular.js front-end and a Rails back-end. - ÖVA is an ongoing project -
