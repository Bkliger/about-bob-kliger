A Little Bit About Me:

I'm a WebDev / Coder with a background in audio engineering and Italian translation. While looking for ways to expand my reach as an audio engineer, I began programming in order to create a game sound portfolio. I caught the coding bug and spent the next year self educating in C# and .NET technologies, along with basic HTML, CSS and Javascript. I joined General Assembly in order to deepen my web based knowledge and experience, and to codify my strategies as I pivot from music and translations to a full-time position as a web developer. Skills: languages - Javascript, Ruby, C#, HTML, CSS, EJS, ERB, Italian. Frameworks - Node, Ruby on Rails, Express, Angular, APS.NET MVC. Technologies: jQuery, Bootstrap, Git/Github, Bcrypt, Underscore.js, Rspec, AJAX, Socket.io, Jasmine, Mongoose, 3rd Party APIs. Databases: MongoDB, PostreSQL, MS SQL Server.

Projects

Twitter-Lounge

Twitter-Lounge is a real time social Twitter experience. The Socket.io powered chat client is accompanied by a configurable live Twitter stream. This project was built using the Node.js/Express/MongoDB stack, with a lightweight jQuery front-end.

Vagabond

Vagabond is a social travel blogging site built using Ruby on Rails. Maps are handled using the Google Maps API, and keywords are extracted from the users posts through the text analysis feature of IBM's Alchemy API.

MoodZiq

This playlist builder and audio player allows the user to pick a duration and fill that time with the musical genre or mood of their choice. Playlists can be saved and recalled through a clean and intuitive interface. MoodZiq was built using Ruby on Rails and JQuery, and draws its tunes from the SoundCloud API.

ÖVA

ÖVA is a run coaching application. An athlete enters their planned weekly training mileage and the days of the week they plan to run, and ÖVA generates a plan based on previous efforts which are gathered through tight integration with Strava. This project was build using an Angular.js front-end and a Rails back-end.
